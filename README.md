# 🗺️ 云旗 & 郝熠然 · 行程地图

云旗和郝熠然的行程轨迹可视化地图，记录两人从相识到一路走来的每个足迹。

**在线访问：** https://penapinner.github.io/yunyi-map/

---

## ✨ 功能介绍

### 🗺️ 交互式地图
- 行程路线以**弧线虚线**展示飞行/乘车路径，起点终点用彩色圆点标记
- 事件地点以**圆点**标记在地图上
- 点击标记可查看详情、跳转日历或打开相关链接
- 地图底图采用 CARTO Light 明亮风格

**颜色说明：**
- 🟠 **橙色** — 云旗单人行程/事件
- 🔴 **红色** — 郝熠然单人行程/事件
- 🔵 **青色** — 双人行程/事件

### 📅 日历视图
- 支持切换**列表**和**日历**两种视图
- 日历上方显示**中国农历**日期和传统节日（春节、元宵、中秋等）
- 有行程记录的日期会显示彩色圆点标识
- 支持按人物筛选（全部 / 云旗 / 郝熠然 / 双人）
- 月份切换 +「今天」快捷跳转

### 📊 数据统计
- **旭日图**：展示云旗、郝熠然、双人行程占比，内层区分路线和事件
- **月度堆叠柱状图**：每月各人物出行次数分布
- **城市排行横版柱状图**：去得最多的城市 TOP 10
- 统计面板默认展开，点击顶栏「统计」按钮可收起/展开

### ⏱️ 时间拉杆
- 地图下方双滑块时间拉杆，可筛选指定时间范围
- 拖动后地图、列表、日历、统计图表**全部联动更新**
- 日历中超范围的日期自动变灰
- 点「重置」恢复全部时间范围

### 🔗 联动功能
- **日历 → 地图**：点击日历日期，地图自动飞到对应位置
- **地图 → 日历**：点击弹窗「日历」按钮，自动跳转并选中日期
- **列表 → 地图**：点击行程卡片，地图飞到对应位置并高亮

### 💬 评论系统
- 基于 **giscus**（GitHub Discussions）的留言板
- 每条行程/事件有独立评论区
- 需登录 GitHub 账号发表评论

## 📋 数据说明

### 数据来源
行程数据来源于公开网络信息整理，包括：
- 微博等社交媒体公开内容
- 公开活动信息与行程报道
- 粉丝社区整理的公开行程记录

### 数据类型
| 类型 | 说明 | 地图展示 |
|------|------|----------|
| **路线 (trip)** | 带起点终点的移动记录（飞行、乘车等） | 弧线 + 起止圆点 |
| **事件 (event)** | 特定地点发生的事件（直播、拍摄、活动等） | 圆点标记 |

### 时间跨度
2025 年 4 月 ~ 2026 年 4 月（持续更新中）

### 数据统计
- 双人行程：44 条
- 郝熠然单人：13 条
- 云旗单人：7 条
- 合计：**64 条记录**

## 🔄 数据更新

行程数据写在 `index.html` 文件内的 `allItems` 数组中。

**更新方式：** 直接告诉 AI 助手要添加/修改/删除的行程信息，助手会自动更新 HTML 文件中的数据，替换到 GitHub 后自动生效。

也可以使用本地编辑器 `editor.html` 进行可视化编辑。
