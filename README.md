# 🗺️ yunyi-map

云旗 & 郝熠然的行程地图可视化

## 访问地址

https://penapinner.github.io/yunyi-map/

## 功能

- 📍 交互式地图展示行程路线
- 🟠🟥🟦 区分云旗/郝熠然/双人行程
- 👨‍💼 管理员：添加/删除行程
- 👤 游客：浏览、评论、上传图片

## 数据更新

行程数据写在 `index.html` 文件内的 `trips` 数组中。

**通过对话更新行程：** 直接告诉 AI 助手要添加/修改/删除的行程信息，助手会自动更新 HTML 文件中的数据，推送后自动生效。
