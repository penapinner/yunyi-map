# 🗺️ yunyi-map

云旗 & 郝熠然的行程地图可视化

## 访问地址

https://penapinner.github.io/yunyi-map/

## 功能

### 📍 地图展示
- 交互式地图展示行程路线（弧线虚线飞行路径）和事件地点（圆点标记）
- **橙色** = 云旗单人行程/事件
- **红色** = 郝熠然单人行程/事件
- **青色** = 双人行程/事件
- 点击地图标记查看详情、跳转日历或相关链接

### 📅 日历视图
- 侧边栏支持**列表**和**日历**两种视图切换
- 日历上有记录的日期显示彩色圆点标识
- 支持按人员筛选（全部 / 云旗 / 郝熠然 / 双人）
- 月份切换 + 「今天」快捷跳转

### 🔗 日历与地图联动
- **日历 → 地图**：点击日历日期，地图自动飞到该日所有行程/事件位置
- **地图 → 日历**：点击地图弹窗中的「日历」按钮，自动跳到日历并选中日期
- **列表 → 日历**：行程卡片上的「日历」按钮可快速跳转
- 选中日期后，日历下方展示该日所有记录详情

### 📋 列表视图
- 按时间倒序展示所有行程路线和事件地点
- 显示统计信息（路线数 / 事件数）
- 支持按人员筛选

### 💬 评论系统
- 基于 **giscus**（GitHub Discussions）的留言板
- 每条行程/事件有独立的评论区
- 访客需登录 GitHub 账号即可发表评论
- 评论数据存储在仓库 Discussions 中，网页更新不影响评论

### 🎨 界面风格
- 明亮简洁的界面设计
- CARTO Light 明亮地图底图

### 🔗 外部链接
- 部分行程/事件附带相关微博等链接，可直接跳转查看

## 数据说明

### 数据来源
行程数据来源于「地图书」(ditushu.com) 导出的 GeoJSON 文件，包含：

| 文件 | 内容 | 数量 |
|------|------|------|
| 郝熠然-2026-04-27.json | 郝熠然单人行程和事件 | 13 条 |
| 云旗-2026-04-27.json | 云旗单人行程和事件 | 7 条 |
| 双人行程-2026-04-27.json | 双人行程和事件 | 44 条 |

### 数据类型
- **路线 (trip)**：带起点终点的移动记录（如飞行、乘车），在地图上显示为弧线 + 起止标记
- **事件 (event)**：特定地点发生的事件（如直播、拍摄、聚餐），在地图上显示为圆点标记

### 时间跨度
2025年9月 ~ 2026年4月（持续更新中）

## 数据更新

行程数据写在 `index.html` 文件内的 `allItems` 数组中。

**通过对话更新行程：** 直接告诉 AI 助手要添加/修改/删除的行程信息，助手会自动更新 HTML 文件中的数据，推送到 GitHub 后自动生效。

## 技术栈

- [Leaflet](https://leafletjs.com/) — 地图引擎
- [CARTO](https://carto.com/) — 地图底图
- [giscus](https://giscus.app/) — 评论系统
- [Font Awesome](https://fontawesome.com/) — 图标
- [Noto Sans SC](https://fonts.google.com/noto/specimen/Noto+Sans+SC) — 字体
